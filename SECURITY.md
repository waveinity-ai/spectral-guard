# Security Policy

## Supported Versions
| Version | Supported |
|---------|-----------|
| 2.0.x   | ✅ Yes    |
| 0.x.x   | ❌ No     |

## Reporting a Vulnerability
Email: security@waveinity.com
Response time: 48 hours
Do NOT open public issues for security vulnerabilities.

## Scope
This library processes dynamic knowledge graph data in highly regulated, mission-critical industries. We treat all security and topological integrity reports with the highest priority.
